# Nexus Command Center

Personal AI-powered dashboard agent task repo.

## Agent Workflow

1. Read `WHITEBOARD.md`
2. Pick one task
3. Read `tasks/T-XXX.md`
4. Implement on local system (`/tmp/nexus-command-center`)
5. Update `WHITEBOARD.md`, commit, push

## Files

- `WHITEBOARD.md` — Task board (source of truth)
- `AGENTS.md` — Agent manifest (read at start)
- `AGENT_GUIDELINES.md` — Rules (read every cycle)
- `tasks/` — Individual task specs

## Repo

Main repo lives on **this machine** at `/tmp/nexus-command-center`. GitHub is the backup mirror.
