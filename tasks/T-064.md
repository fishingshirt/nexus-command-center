# T-064 — Agent Virtual Office (Pixel Art Game View)

> **Status:** `PENDING` — user request, do not start yet  
> **Priority:** LOW (exploratory / creative feature)  
> **Depends on:** None — greenfield feature

---

## 📋 Description

A new **Agents** category/tab in the Nexus Command Center that displays all active Hermes agents in a **virtual pixel-art office environment** — styled like a retro game.

Each agent is represented as a **pixelated character** (similar to NPCs in classic RPGs). The office shows:
- **Working agents** → at their desks, with a speech bubble or status label showing what task they're on
- **Idle agents** → hanging out by a **water cooler** (or coffee machine, lounge area, etc.)
- **Status indicators** — maybe a "busy" animation vs. "idle" bounce

This is a **purely visual, gamified dashboard view** — not a functional task manager, but a fun way to see agent activity at a glance.

---

## 🎨 Creative Direction

- **Style:** 16-bit / 8-bit pixel art aesthetic (think Stardew Valley, Habbo Hotel, or classic RPG towns)
- **Setting:** Small office / command center room with desks, chairs, a water cooler, maybe a whiteboard on the wall
- **Agents:** Each agent gets a tiny pixel avatar (could be auto-generated from initials + color, or use a small sprite set)
- **Animation:** Subtle idle animations (breathing bounce, coffee sip, typing wiggle)
- **Interactivity:** Click an agent to see their current task details in a popup modal
