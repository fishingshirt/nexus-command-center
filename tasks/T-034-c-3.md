# T-034-c-3 — Conflict Resolution Diff Panel UI

## Goal
When the user clicks a conflicted event chip, show a side-by-side diff overlay letting them choose which version wins.

## Acceptance Criteria
- [ ] Clicking a conflicted event chip opens a new overlay `#conflict-panel` (not the normal event modal).
- [ ] Overlay shows two columns:
  - Left: local version (title, date, start/end time, description)
  - Right: Google version fetched fresh via `GET /api/calendar/events/{gcalId}`
- [ ] Two buttons at the bottom: **"Keep Mine"** and **"Use Google"**.
- [ ] "Keep Mine" → PATCH Google Calendar with local data, then clear `conflict: true`.
- [ ] "Use Google" → Overwrite local event with Google data, then clear `conflict: true`.
- [ ] After resolution, re-render calendar and close panel.
- [ ] Escape key or backdrop click closes the panel with no changes.

## Files to Touch
- `public/js/apps/calendar.js` — add `#conflict-panel` markup injector, click handler for conflict chips, fetch + resolve logic.
- `public/css/apps/calendar.css` — diff panel overlay styles (dark overlay, side-by-side cards, amber border).
