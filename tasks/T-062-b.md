# T-062-b: Recipe App — CSS Module

## Goal
Style the Recipe App.

## Acceptance Criteria
1. Create `public/css/apps/recipe.css`:
   - `.recipe-grid`: responsive grid of cards (1 col mobile, 2 tablet, 3+ desktop)
   - `.recipe-card`: rounded, image top, title, meta row (stars + time)
   - `.recipe-detail`: scrollable, ingredient checkboxes, numbered steps
   - `.recipe-form`: clean inputs, dynamic add/remove buttons for ingredients/steps
   - `.cooking-mode`: full-screen overlay, large text, step progress bar
   - Shopping list: compact rows with checkbox + quantity
   - Dark mode compatible via CSS custom properties
2. Import in `public/index.html` head.

## Files Expected to Touch
- `public/css/apps/recipe.css` (new)
- `public/index.html`
