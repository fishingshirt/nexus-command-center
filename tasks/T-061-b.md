# T-061-b — Wishlist App: CRUD (Add, Edit, Delete)

> **Parent:** T-061 Wishlist App  
> **Priority:** MEDIUM  
> **Status:** PENDING

---

## Scope

Implement the full data model + CRUD operations for wishlist items. This includes the add-item modal, edit modal, delete confirmation, and rendering the grid of cards.

---

## Acceptance Criteria

- [ ] Add Item modal (title, URL, price, currency, priority, tags, notes, image URL)
- [ ] Edit item modal (pre-populated fields)
- [ ] Delete item with confirmation toast
- [ ] Items render as cards in `.wishlist-grid`
- [ ] Card shows: title, price badge, priority color strip, status pill, favicon
- [ ] `localStorage` key `ncc-wishlist-v1` stores `{ items: [...] }`
- [ ] Status transitions: Want → Watching → Purchased → Archived
- [ ] Archived items hidden by default (toggle)
