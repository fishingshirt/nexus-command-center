# T-057: Default Theme Should Be Dark

## Status
`PENDING`

## Goal
The dashboard currently loads the "professional" theme (light mode) by default. Change this so the default theme is a dark theme.

## Acceptance Criteria
- [ ] Change default `<link rel="stylesheet" href="css/themes/professional.css" id="theme-stylesheet">` in `index.html` to load a dark theme by default (e.g., `jarvis.css` or `midnight-hacker.css`)
- [ ] Update the default value in settings (`public/js/settings.js`) so new users start with a dark theme selected
- [ ] Update `theme-color` meta tag in `index.html` to match the dark theme base color
- [ ] Ensure `localStorage` theme key initialization defaults to the dark theme name
- [ ] The welcome overlay and onboarding should also render in the dark theme by default
- [ ] If the user has already saved a theme preference in localStorage, respect it (do not override on every load)

## UI Details
- Default dark theme suggestion: `jarvis.css` (Nexus branding with dark charcoal + electric blue)
- Meta theme-color should match --bg-body of the chosen default theme

## Files Expected to Touch
- `public/index.html`
- `public/js/settings.js`
- `public/js/app.js` (theme initialization logic)

## Dependencies
- None
