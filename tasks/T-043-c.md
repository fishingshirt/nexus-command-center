# T-043-c — Quick Capture: Voice Memo Recording + Storage

## Goal
Add voice memo capture to the Quick Capture widget: hold-to-record, stop, metadata preview, and blob storage.

## Files To Touch
1. `public/js/apps/quick-capture.js` (extends T-043-b)
2. `public/css/quick-capture.css` (adds voice UI styles)

## Acceptance Criteria
- [ ] Satellite "Voice" button opens capture modal switched to voice mode
- [ ] Voice mode shows a microphone icon, "Hold to record" label, and a circular hold area
- [ ] Start recording on `mousedown`/`touchstart`, stop on `mouseup`/`touchend` (or tap toggle for accessibility)
- [ ] Recording state: red pulsing dot, elapsed timer (MM:SS), waveform visualization (optional simplified bar)
- [ ] Use `navigator.mediaDevices.getUserMedia({ audio: true })` → `MediaRecorder` with `audio/webm` or fallback mimetype
- [ ] On stop: create blob URL, show playback audio element, save button, discard button
- [ ] Store audio blob in `IndexedDB` under store `nexus-voice-memos`; keep metadata (`id`, `title`, `duration`, `createdAt`, `blobKey`) in `localStorage` under key `ncc-voice-memos`
- [ ] If `IndexedDB` unsupported, fallback to `localStorage` base64 with quota warning toast if >1 MB
- [ ] Transcript: use `webkitSpeechRecognition` if available; show interim + final transcript below playback
- [ ] Add `#voice-memos-list` view to capture modal for browsing saved memos

## Tech Notes
- `MediaRecorder` check: `if (MediaRecorder.isTypeSupported('audio/webm;codecs=opus')) mimeType = 'audio/webm;codecs=opus'; else mimeType = 'audio/webm'; else fallback`
- IndexedDB wrapper: `openDB('nexus-voice-store', 1)` → objectStore `memos` with keyPath `id`
- Each memo metadata in localStorage: `{ id: crypto.randomUUID(), title: 'Voice Memo [date]', duration: seconds, createdAt: ISO, blobKey: id }`

## Accessibility
- `aria-pressed` on hold button when recording
- `aria-live="polite"` on timer
- Respect `prefers-reduced-motion`: disable pulse animation, use static color instead

## Fallback
- If `navigator.mediaDevices` unavailable (http / insecure context): show message "Microphone requires HTTPS or localhost"
- If `MediaRecorder` unavailable: show "Voice recording not supported in this browser"
