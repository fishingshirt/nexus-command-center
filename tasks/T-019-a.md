# T-019-a: Finance Tracker — JS App Module + Wiring

## Goal
Implement the Finance Tracker app module and wire it into the dashboard shell.

## Acceptance Criteria
1. Create `public/js/apps/finance-tracker.js` with:
   - App lifecycle: `initApp`, `showApp`, `hideApp`
   - State in `localStorage` key `ncc-finance-transactions` (list of objects: `{id, date, amount, category, note, type: 'income'|'expense'}`)
   - Add transaction form: amount (number), type (toggle income/expense), category (select: Food, Transport, Housing, Entertainment, Health, Shopping, Income, Other), note (text), date (date input, default today)
   - Transaction list with delete button per item
   - Monthly summary bar: total income, total expense, net
   - Category breakdown horizontal bar (each category as a segment, colored)
   - CSV export button: generates `nexus-finance.csv` download
2. Wire in `public/index.html`:
   - Add `<div id="view-finance-tracker" class="app-view" aria-label="Finance Tracker">...</div>` inside `<main>`
   - Add nav link `<li><a href="#finance-tracker" data-app="finance-tracker">💰 Finance</a></li>` inside `#app-nav ul`
   - Add `<script type="module" src="js/apps/finance-tracker.js"></script>` after other app scripts
3. Wire in `public/js/app.js` registry:
   - Add `finance-tracker` entry in `appsRegistry` with `name`, `icon`, `init` callback
4. Make sure `initApp('finance-tracker')` is idempotent.

## Files Expected to Touch
- `public/js/apps/finance-tracker.js` (new)
- `public/index.html` (view block + nav link + script tag)
- `public/js/app.js` (registry entry)

## Dependencies
None.
