# T-044 — World Clock Widget

## Goal
A widget and mini-app that displays multiple timezones, useful for calling people in different cities or tracking global markets.

## Acceptance Criteria
- [ ] Add/remove cities: search by name, pick from dropdown
- [ ] Dashboard widget: horizontal row of location pills showing analog or digital time
- [ ] Full app view: grid of cards with city name, country, day/night indicator, offset from local
- [ ] Configurable formats: 12h / 24h, show date, show seconds
- [ ] Day/night visual: sun ☀️ / moon 🌙 icon or gradient background
- [ ] Auto-sync: updates every minute via `setInterval`
- [ ] Agent command: "add Dubai to my clocks", "what time is it in Tokyo"

## UI Details
- Widget row: compact, no labels, just "London 14:30 ☀️", scroll horizontally if >4
- Full card: larger digital clock, analog SVG clock optional, "+3 hours ahead" text
- Settings: reorder drag-and-drop, delete swipe

## Tech Notes
- Pure JS `Date` with `Intl.DateTimeFormat`: `new Intl.DateTimeFormat('en-US', {timeZone: 'Asia/Tokyo', hour: '2-digit', minute: '2-digit'}).format(new Date())`
- Store: `[{id, city, timezone, label}]` in `localStorage`
- Max 8 cities
- Analog clock: SVG `<circle>` + `<line>` for hands, CSS rotate transform
