# T-045 — Pomodoro Timer / Focus Mode

## Goal
A built-in productivity timer with focus sessions, break reminders, and session history.

## Acceptance Criteria
- [ ] Timer modes: Pomodoro (25m work + 5m break), Short Break (5m), Long Break (15m), Custom
- [ ] Full-screen focus mode: minimal timer display, blocks distracting UI elements
- [ ] Session counter: tomatoes 🍅 tracked per day, streak display
- [ ] Notifications: browser alert when session ends, optional sound
- [ ] Task linking: before starting timer, pick which todo item you're working on
- [ ] History: sessions logged with date, duration, linked task, interruptions
- [ ] Agent command: "start a 25 minute focus timer" or "how many pomodoros today"

## UI Details
- Widget: circular progress ring (SVG `stroke-dasharray`), time remaining in center
- Controls: play/pause, skip, reset
- Full-screen focus: black background, large timer, small task text below
- Today stats: "4 sessions, 1h 40m focused"
- Break overlay: "Take a break — step away from the screen"

## Tech Notes
- Timer logic: `setInterval` at 1s, visual update via CSS variable `--timer-progress`
- Sound: Web Audio oscillator gentle chime (no external files)
- Store sessions: `localStorage` `ncc-pomodoro-sessions` array
- Background tab: use `Page Visibility API` to catch up missed seconds
