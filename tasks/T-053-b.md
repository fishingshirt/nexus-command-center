# T-053-b: News Hub — Feed Engine & Offline Cache

## Status
`PENDING`

## Goal
Wire the news list to fetch real headlines from a free news API (NewsAPI or GNews), cache up to 50 articles in `localStorage`, and render them with category/search filtering.

## Acceptance Criteria
- [ ] Server endpoint `GET /api/news` proxies to NewsAPI (or GNews) using a config key.
- [ ] Query params: `category`, `q`, `page`, `pageSize`. Defaults: page=1, pageSize=20.
- [ ] Server-side cache folder `~/.hermes/nexus-news-cache/` with JSON files keyed by `(category,q,page)` and 15-minute TTL.
- [ ] If API key missing / fetch fails → return mock static JSON (fallback).
- [ ] Frontend `news.js` loads articles on `openNews()` and renders cards with thumbnail, title, source, relative time, snippet.
- [ ] `localStorage` cache key `ncc-news-cache` stores last 50 articles with `cachedAt` timestamp (24h expiry).
- [ ] Offline: if fetch fails and cache exists, show cached articles with a subtle "offline" chip.
- [ ] Refresh button in header triggers reload.

## Files Expected to Touch
- `public/js/apps/news.js`
- `scripts/nexus-server.py` (`/api/news` handler)

## Dependencies
