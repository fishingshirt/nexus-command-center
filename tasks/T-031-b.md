# T-031-b: Cloud Vault — CSS + Theme Integration

## Goal
Style the Vault app with the project's design system.

## Acceptance Criteria
1. Create `public/css/apps/vault.css` with:
   - `.vault-container` layout: sidebar folders (25% width desktop) + main file list
   - Upload drop zone: dashed border, highlight on dragover, centered icon + text
   - File list: rows with alternating background, icon + name + size + date
   - Folder tree: nested indent, expand/collapse caret, active highlight
   - Search input: full-width above file list, with clear button
   - Responsive: stack sidebar above list on mobile
   - Dark mode compatible using CSS custom properties (`--bg-body`, `--text-main`, `--border`, etc.)
   - No hardcoded hex colors
2. Import in `public/index.html` (inside head, same pattern as other app CSS):
   - Add `<link rel="stylesheet" href="css/apps/vault.css">`

## Files Expected to Touch
- `public/css/apps/vault.css` (new)
- `public/index.html` (CSS link)

## Dependencies
T-031-a (JS selectors must exist)
