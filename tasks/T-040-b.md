# T-040-b: Dashboard Widget Grid — CSS + Add/Remove Drawer

## Goal
Style the widget grid and build the "Add Widget" drawer UI.

## Acceptance Criteria
1. Create `public/css/apps/widget-grid.css` with:
   - `.widget-grid` container: `display: grid; grid-template-columns: repeat(auto-fill, minmax(280px, 1fr)); gap: var(--space-4);`
   - `.widget-card` style: rounded rectangle, app-colored top border, title bar with icon, refresh spinner
   - `.widget-edit-overlay` dim background when in edit mode
   - Responsive: 1 column at <768px, 2 at 768-1200px, 3-4 above
   - Edit mode drag handle and delete button styling
   - `.widget-drawer` slide-up panel for adding widgets
2. In `public/js/widgets/grid.js` (or new `drawer.js`):
   - "Add Widget" button on dashboard header opens drawer
   - Drawer lists all registered widgets with preview name
   - Tap to add; default position is end of layout
   - Empty state on dashboard: "Tap + to add your first widget"

## Files Expected to Touch
- `public/css/apps/widget-grid.css` (new)
- `public/index.html` (drawer markup if needed)
- `public/js/widgets/grid.js` (add drawer logic)
