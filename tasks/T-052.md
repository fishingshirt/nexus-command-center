# T-052 — Project Completion Sentinel

## Goal
When every feature task and bug fix in the active queue is marked `DONE` (leaving only this sentinel task), pause the automated agent cronjob, notify the user via Telegram, and wait for human review before resuming any work.

## Acceptance Criteria
- [ ] **Done-detection logic**: Agent evaluates WHITEBOARD.md on each wake. If all tasks except T-052 are `DONE`, proceed with shutdown sequence.
- [ ] **Pause cronjob**: Disable the `nexus-build-agent` cron job (`~/.hermes/scripts/nexus-build-agent.sh` or via `hermes cronjob edit 3e69e5f54c6e`) to stop automated wake cycles.
- [ ] **Notify user**: Send Telegram direct message:  
  `\u2705 Nexus build complete! All tasks DONE. Agent paused. Please review the dashboard and reply if you spot any issues or want to add new tasks.`
- [ ] **Wait state**: Agent enters a passive mode. No automated execution until:
  - User adds new tasks to WHITEBOARD.md, **or**
  - User explicitly replies "resume" / "continue" / "unpause"
- [ ] **Resume on demand**: If user requests resume, re-enable the cronjob and restore normal wake cycles.
- [ ] **WHITEBOARD update**: Mark T-052 as `DONE` only after the pause-and-notify sequence is fully executed and confirmed.

## UI / Dashboard
- Add a banner to the Agent Stats panel when the cronjob is paused:  
  `\ud83d\udee1\ufe0f Agent paused — waiting for your review.`
- Persist pause state in `localStorage` (`ncc-agent-paused: true`) so the UI reflects it across reloads.

## Tech Notes
- Cronjob ID reference: `3e69e5f54c6e` (Hermes cron task running every 15 minutes).
- Telegram target: `telegram:203879215` (Jay DM).
- The pause is **only** triggered when T-052 is the sole remaining non-done task. If user adds new tasks before T-052 triggers, normal priority rules apply.

## Dependencies
- All other active tasks must be `DONE` before this activates.
- Should be the **last** task created in this phase.

## Agent Note
This is the "finished" signal. Do not over-automate after this point — the user wants manual control. Do not unpause unless explicitly instructed.
