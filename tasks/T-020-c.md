# T-020-c: AI Task Suggester — Server API Endpoint

## Status
`PENDING`

## Goal
Add `/api/ai/suggest` POST endpoint that calls the local Ollama model.

## Acceptance Criteria
1. In `scripts/nexus-server.py`:
   - Add `_api_ai_suggest(handler, path)` helper inside `do_POST()` dispatch
   - Accept `POST /api/ai/suggest`
   - Parse JSON body: `{ context: { recentEvents[], pendingTodos[], notesCount } }`
   - Build prompt for `_ollama_generate()`:
     ```
     You are a productivity assistant. Given the user's recent dashboard data, suggest 3-5 actionable tasks or improvements.
     Respond ONLY in valid JSON with no markdown formatting. Format:
     { "suggestions": [
       { "id": "suggest-1", "title": "...", "description": "...", "priority": "HIGH|MEDIUM|LOW", "category": "work|personal|health|learning|chores" }
     ]}
     ```
   - Call `_ollama_generate(prompt)` with fallback if Ollama is not running
   - Return `{ ok: true, suggestions: [...] }` or `{ ok: false, error: string }`
   - Ensure `_api_ai_suggest` is registered in `do_POST()` dispatch chain
