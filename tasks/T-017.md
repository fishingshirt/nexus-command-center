# T-013: Data Persistence Layer

## Status
`PENDING`

## Goal
Migrate all app data from `localStorage` to a real server-side JSON/SQLite store.

## Acceptance Criteria
- [ ] All `localStorage` reads/writes go through a `StorageAdapter`
- [ ] Server-side JSON or SQLite file stores app data per user session
- [ ] Dashboard still works offline (fallback to localStorage)
- [ ] Data survives browser cache clear

## Files Expected to Touch
- `nexus-server.py` (new endpoints `/api/data/{app}/read`, `/api/data/{app}/write`)
- `public/js/app.js` (StorageAdapter wrapper)

## Dependencies
T-012 (Hermes bridge) can run in parallel.
