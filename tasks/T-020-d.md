# T-020-d: AI Task Suggester — Settings Integration

## Status
`PENDING`

## Goal
Add settings toggles for the AI Suggester.

## Acceptance Criteria
1. In `public/js/settings.js` (or existing settings panel):
   - Add "AI Suggester" section
   - Toggle: "Enable AI suggestions" (default ON)
   - Dropdown: "Model" (`llama3.2`, `llama3`, `mistral`) — stored in `localStorage`
   - Dropdown: "Frequency" (`hourly`, `daily`, `weekly`, `manual`) — stored in `localStorage`
   - Button: "Clear dismissed suggestions"
   - Persist settings to `localStorage` key `ncc-ai-suggester-settings`
2. In `public/js/apps/ai-suggester.js`:
   - Read settings on init; if disabled, show a message "AI Suggester is disabled in Settings" instead of fetching
   - Respect frequency: only auto-fetch if frequency is not `manual`

- `public/js/apps/ai-suggester.js`
