# T-042-b — Notification Center UI (Bell, Badge, Slide-out Panel)

## Goal
Build the notification center user interface: bell icon in header, unread badge, slide-out panel with grouped list, dismiss controls, and empty state.

## Acceptance Criteria
- [ ] Bell icon added to `.header-right` in `index.html` (left of search button)
- [ ] Red badge with white number over bell; hidden when count=0
- [ ] Click bell → slide-out panel from right side
- [ ] Panel header: "Notifications" + gear icon → settings + "Clear all" link
- [ ] Panel body: grouped by relative date (Today / Yesterday / Earlier)
- [ ] Each row: app emoji icon, title, body (2-line clamp), relative time, dismiss ❌
- [ ] Unread items have blue dot left of title; click row to mark read
- [ ] Empty state: "All caught up 🎉"
- [ ] Escape or click backdrop closes panel
- [ ] Focus trap inside panel while open

## Files to Touch
1. `public/index.html` — bell button + notification panel markup (before closing `</body>`)
2. `public/css/base.css` — `.notification-bell`, `.notification-badge`, `.notification-panel`, `.notification-item`, etc.
3. `public/js/notifications.js` — add `initNotifications()` function that wires panel, badge, list rendering, open/close, mark-read, dismiss

## CSS Notes
- Panel: `position: fixed; inset: 0; z-index: 90;` overlay + right-side sheet (`width: min(380px, 90vw)`)
- Backdrop: semi-transparent black
- Sheet: `background: var(--surface); border-left: 1px solid var(--border);`
- Smooth slide via `transform: translateX(100%)` → `translateX(0)`; wrap in `@media (prefers-reduced-motion: no-preference)` or provide reduced-motion instant fallback
- Badge: `position: absolute; top: -2px; right: -2px;` inside bell button

## JS Notes
- `initNotifications()` listens for `nexusNotification` events to refresh badge count and, if panel open, prepend new item
- Uses existing `getNotifications()`, `markRead()`, `dismiss()`, `clearAll()` from `notifications.js`
- Relative time helper: `"2 min ago"`, `"1 hour ago"`, `"Yesterday"`, `"3 days ago"`
- Gear icon jumps to `#settings`
- On bell click, add `.open` to panel; add `panel-open` class to body to prevent background scroll
- Cleanup: `Escape` key listener, backdrop click

## Sub-Task Boundary
This sub-task is UI only. The wiring of calendar + todo to emit notifications (T-042-c) and agent posting endpoint (T-042-d) are separate sub-tasks.
