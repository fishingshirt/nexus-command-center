## UI Details

### Search / Sort / Filter Bar
- Placed below view-header, inside view-body before grid.
- Row layout: Search input (flex:1), Sort select, Filter select, archived toggle checkbox.
- Mobile: stack vertically on ≤480px.

### Edit Modal
- Reuses existing `#wishlist-modal` and `#wishlist-form`.
- Sets `data-editing-id` on form to track active edit.
- `openEditModal(id)` populates form fields from item.
- `onFormSubmit()` checks `data-editing-id` — if present, updates existing item.

### Search Logic
- Real-time filtering on `input` event.
- Matches title, notes, or any tag (case-insensitive).
- Displays "N results" count.

### Sort Logic
- Options: Price (low → high, high → low), Priority (Must Buy → Low), Status (Want → Archived), Created (newest, oldest), Updated (newest, oldest).
