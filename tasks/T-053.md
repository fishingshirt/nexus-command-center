# T-053: News & Media Hub

## Status
`PENDING`

## Goal
Replace the legacy RSS Reader concept with a full-featured News & Media Hub that pulls live news by topic, filters by category, embeds suggested YouTube videos, and offers an AI-powered daily digest toggle.

## Acceptance Criteria
- [ ] **News Feed Engine**
  - Fetch headlines from a free news API (e.g., NewsAPI, GNews, or RSS-to-JSON proxy).
  - Default categories: World, Politics, Technology, Business, Sports, Entertainment, Science, Health.
  - Category filter tabs / chips at the top of the app.
  - Search bar to query keywords across headlines.
  - Article cards: thumbnail image, headline, source, relative timestamp (e.g., "2h ago"), brief snippet.
  - Tap/click article → opens inline reader overlay or external link.
  - Pull-to-refresh (mobile) and manual refresh button.
  - Offline cache: last 50 articles stored in `localStorage` with expiry (24h).

- [ ] **YouTube Daily Suggestions**
  - Integrate YouTube Data API v3 or use a curated `/api/youtube/daily` server endpoint.
  - Display 4–6 suggested videos in a horizontal scroll strip below the news feed.
  - Video cards: thumbnail, title, channel name, duration.
  - Tap → opens inline YouTube embed or native link.
  - "Refresh suggestions" button to load a new batch.
  - Categorize suggestions: Trending, Tech, Entertainment, etc.

- [ ] **AI Daily Digest Toggle**
  - Settings toggle: `Enable AI Daily Digest` (default OFF).
  - When ON, agent fetches top headlines + selected categories on wake.
  - Agent posts a formatted summary to the News Hub (or to a dedicated "Digest" panel):
    - Key headlines grouped by category.
    - Short 1-sentence summary per article (LLM-generated or API-extracted).
    - Highlighted "Top story of the day".
    - Optional: "You may also like" cross-suggestions from YouTube.
  - Digest panel appears at the top of the News Hub when available.
  - User can dismiss individual digest cards.
  - Last digest timestamp shown.

- [ ] **Dashboard UI**
  - Full app shell matching Nexus theme (`index.html` iframe slot or injected view).
  - Header: app icon, title "News Hub", settings gear, refresh button.
  - Filter chip bar: All | World | Politics | Tech | Business | Sports | Entertainment | Science | Health.
  - Active chip highlighted with accent color; inactive chips dimmed.
  - News list: masonry or vertical card stack (mobile-optimized).
  - YouTube strip section header: "Suggested for You" with refresh icon.
  - Digest banner at top when AI summary is present (collapsible).
  - Skeleton loaders while fetching.
  - Empty state: "No articles found. Check connection."
  - Error state: "Failed to load news. Retry?"

- [ ] **Server API Endpoints** (extend `nexus-server.py`)
  | Endpoint | Method | Description |
  |----------|--------|-------------|
  | `/api/news` | GET | Query params: `category`, `q`, `page`, `pageSize`. Returns `{articles[], totalResults}`. Caches 15 min server-side. |
  | `/api/news/categories` | GET | Returns `['world', 'politics', ...]`. |
  | `/api/youtube/daily` | GET | Returns curated `{videos[]}` from YouTube API or pre-seeded list. |
  | `/api/news/digest` | POST | Body `{categories[], maxArticles}` → triggers agent LLM summary generation. Returns `{digestId, status}`. |
  | `/api/news/digest/:id` | GET | Poll for completed digest `{ready: bool, content: {...}}`. |

- [ ] **Settings Schema Addition**
  ```json
  "newsHub": {
    "enabled": true,
    "defaultCategory": "all",
    "aiDigestEnabled": false,
    "digestCategories": ["world", "politics", "technology"],
    "youtubeEnabled": true,
    "youtubeCategories": ["trending", "technology"],
    "articleCacheHours": 24,
    "apiKey": null
  }
  ```

- [ ] **Agent Integration**
  - When `aiDigestEnabled` is true and the agent wakes, check if last digest is > 4 hours old.
  - If stale, agent calls `/api/news/digest` with user-selected categories.
  - Agent waits for digest completion, then stores result.
  - Notifications center gets a toast: "📰 Your daily news digest is ready."
