# T-017-a: Spotify Integration — OAuth + Playback Shell

## Goal
Connect Spotify OAuth and build basic playback UI.

## Acceptance Criteria
1. In `scripts/nexus-server.py`:
   - `GET /api/spotify/oauth/start` — redirect to Spotify auth with PKCE (or state param)
   - `GET /api/spotify/callback` — exchange code for tokens, save to `data/spotify-tokens.json`
   - `GET /api/spotify/status` — `{linked:bool, userName?}`
2. In `public/js/apps/spotify.js`:
   - Auth flow: button opens `/api/spotify/oauth/start`
   - Once linked, show "Now Playing" card: track name, artist, album art, play/pause, next/prev
   - Poll `/api/spotify/now-playing` every 5s (server proxies Spotify API)
   - Playlist browser: list user's playlists, tap to queue
3. Wire in `public/index.html` and `public/js/app.js`.
4. Add `.gitignore` rule for `data/spotify-tokens.json`.

## Files Expected to Touch
- `scripts/nexus-server.py`
- `public/js/apps/spotify.js` (new)
- `public/index.html`
- `public/js/app.js`
- `.gitignore`

## Dependencies
