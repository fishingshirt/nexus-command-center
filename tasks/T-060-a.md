# T-060-a — QC Auto Gate: Syntax & Cross-Reference Validator

## Status
`PENDING`

## Goal
Build the Stage 1 automated quality check script (`scripts/nexus-qc.py`). This is the FIRST sub-task and must be completed before any other QC/QA pipeline work.

## Acceptance Criteria
- [ ] `scripts/nexus-qc.py` created and executable
- [ ] `check_syntax_js(dir)` — validates all `.js` files via `node --check` or `acorn.parse`. Returns list of files pass/fail with line numbers.
- [ ] `check_syntax_css(dir)` — lightweight regex/CSS parser to catch unclosed braces, missing semicolons on rule endings (at minimum)
- [ ] `check_html_wellformed(dir)` — validates `index.html` has balanced tags and critical IDs exist
- [ ] `check_cross_reference(html_path, js_dir, css_dir)` — verifies every `getElementById('X')` has matching `id="X"`, every `querySelector('.X')` has matching `.X` in CSS, and every `data-app` has a matching `#view-app`
- [ ] `check_server_routes(server_path, js_dir)` — greps all `/api/` strings in JS and confirms they exist in `nexus-server.py`
- [ ] `check_security_scan(files)` — flags hardcoded secrets, suspicious `eval/innerHTML` patterns, unescaped inputs
- [ ] Writes result JSON to `~/.hermes/nexus-quality-queue.json`
- [ ] Exit code 0 = pass, exit code 1 = fail (for CI/githook integration)
- [ ] Logs to `~/.hermes/nexus-audit-log.ndjson` (append-only)

## Output Format
```json
{
  "id": "QC-001",
  "timestamp": "2026-05-10T12:00:00Z",
  "commit": "HEAD",
  "files_changed": ["public/js/apps/weather.js"],
  "checks": [
    {"check": "syntax-js", "status": "pass", "detail": "15 files OK"},
    {"check": "crossref-ids", "status": "FAIL", "detail": "getElementById('x') missing id='x' in index.html at line 47"}
  ],
  "overall": "FAIL"
}
```

## Tests
- Run against current repo → should catch the existing unreferenced IDs
- Run with intentionally broken JS → must fail
- Run with valid code → must pass

## Agent Note
Keep the script self-contained. Use only Python standard library + `subprocess` for Node calls. Do NOT add npm dependencies.
