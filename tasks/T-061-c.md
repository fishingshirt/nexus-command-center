# T-061-c — Wishlist App: Auto-fetch Link Metadata

> **Parent:** T-061 Wishlist App  
> **Priority:** MEDIUM  
> **Status:** PENDING

---

## Scope

When a user pastes a URL into the add-item form, attempt to auto-fetch title, image, and price via a server-side proxy endpoint.

---

## Acceptance Criteria

- [ ] Server endpoint `/api/fetch-link` accepts `?url=` and returns JSON `{title, image, price, description}`
- [ ] Frontend calls proxy on URL paste (with debounce)
- [ ] Falls back gracefully on failure (user can still type manually)
- [ ] Uses `fetch` with `no-cors` fallback message
- [ ] No external API keys required

---

## Files
- `scripts/nexus-server.py` (new endpoint)
- `public/js/apps/wishlist.js` (paste handler)
