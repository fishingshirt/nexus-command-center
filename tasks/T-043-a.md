# T-043-a — Quick Capture: FAB + Radial Menu + Modal Shell

## Goal
Add a floating action button (FAB) on the dashboard and every app view, plus the expansion radial menu, plus the quick capture modal markup + CSS.

## Files To Touch
1. `public/index.html` — add FAB element, radial menu, quick capture modal, JS module script tag
2. `public/css/quick-capture.css` — FAB styles, radial menu animation, capture modal

## Acceptance Criteria
- [ ] FAB: fixed bottom-right (16px from edges on mobile, 24px on desktop), circle 56px diameter, solid `--accent` color, white `+` icon
- [ ] FAB expands on click: 4 satellite buttons arranged in a semi-circle (note, event, todo, voice) with icons + labels (icon top, label bottom)
- [ ] Each satellite button triggers the capture modal with a specific `data-capture-mode` attribute
- [ ] Capture modal: fixed position, slides up from bottom, dark backdrop, modal panel with header + fields + save/cancel buttons, max-width 640px, centered on desktop
- [ ] Modal markup contains field containers for all capture types: note textarea, event title + date + time, todo text, voice recorder UI
- [ ] Keyboard shortcut: `Ctrl+Space` or `Cmd+Space` toggles FAB open/closed
- [ ] Mobile swipe-up from bottom edge (touchstart at y > window.innerHeight - 40, deltaY < -60) opens capture modal directly in "note" mode
- [ ] CSS variables for timing: `--fab-menu-open: 0.25s`, `--modal-slide-up: 0.2s`, `--backdrop-fade: 0.15s`
- [ ] Wrap all animations in `@media (prefers-reduced-motion: no-preference)`
- [ ] `aria-label` on all icon-only buttons; `role="dialog"`, `aria-modal="true"` on capture modal
- [ ] `z-index` ordering: FAB > nav > backdrop > modal

## HTML Placement
- Insert FAB + radial menu as direct children of `<body>`, after the `#search-overlay` block, before `<script>` tags
- Insert capture modal as another direct child of `<body>`
- Add module script: `<script type="module" src="js/apps/quick-capture.js"></script>` after the existing app scripts and before `app.js`

## CSS Requirements (Mobile-First)
- `@media (min-width: 768px)`: FAB 64px diameter, satellite buttons larger labels, modal full-width-ish with rounded top corners
- `@media (min-width: 1024px)`: modal max-width 640px, centered by default

## Pitfall Avoidance
- Do NOT use more than one line break between elements in the HTML or the patch will fail
- Do NOT start the server before committing this spec-only change
