# T-062-a: Recipe App — JS App Module + Wiring

## Goal
Build the Recipe App frontend.

## Acceptance Criteria
1. Create `public/js/apps/recipe.js`:
   - State: `ncc-recipes` in localStorage (list of `{id, title, ingredients[], steps[], rating, cookTime, servings, notes, tags[], createdAt}`)
   - Recipe list: search, sort (rating, time, recently added), filter by tag
   - Recipe card: image placeholder, title, rating stars, cook time, tags
   - Recipe detail view: expandable, shows ingredients (checkbox to mark "have"), steps (numbered), notes, rating editor
   - Add/edit form: title, ingredients (dynamic list), steps (dynamic list), cook time, servings, tags, notes
   - Delete recipe
   - Cooking mode: step-by-step full-screen, tap to advance, timer per step (optional)
   - Shopping list: from any recipe, "Add to shopping list" → accumulate items in `ncc-recipe-shopping`
2. Wire in `public/index.html`: view block, nav link, script tag.
3. Wire in `public/js/app.js`: import + init + registry.

## Files Expected to Touch
- `public/js/apps/recipe.js` (new)
- `public/index.html`
- `public/js/app.js`

## Dependencies
None
