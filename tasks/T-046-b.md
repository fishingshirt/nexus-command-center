# T-046-b: Auto Dark Mode — Settings Panel + Per-App Override

## Goal
Add settings toggles for auto dark mode and per-app theme overrides.

## Acceptance Criteria
1. Update `public/js/settings.js`:
   - Add section "Auto Theme" with:
     - Toggle: "Follow system theme"
     - Toggle: "Auto-switch by time" + time inputs for dark start and light start
     - Slider: "Transition duration" (0-1s)
   - Add section "Per-App Themes": list all apps with a dropdown to force a specific theme (or "Use global")
   - Settings keys: `themeMode`, `darkStart`, `lightStart`, `themeTransitionMs`, `perAppThemes` (JSON object)
2. Update `public/index.html` settings view markup if needed.
3. Ensure `theme-autoswitch.js` reads settings via `loadSettings()` and applies them.
4. Per-app override: when switching apps, if `perAppThemes[app]` is set, temporarily swap the theme CSS link `href` to that theme, then restore on leaving.

## Files Expected to Touch
- `public/js/settings.js` (new toggles + time inputs + slider + per-app dropdowns)
- `public/index.html` (settings view markup additions)
- `public/js/theme-autoswitch.js` (settings wiring)
